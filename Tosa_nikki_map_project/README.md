# Tosa Nikki Map Project

Questo progetto di Digital Public History ricostruisce il viaggio di Ki no Tsurayuki raccontato nel *Tosa Nikki*.

## Cosa contiene

- Estrazione delle tappe dal testo originale in giapponese
- Conversione dei toponimi antichi in nomi moderni (con chatGPT)
- Geolocalizzazione con Geopy
- Mappa interattiva generata con Folium
- Pagina web HTML + CSS con contenuti storici basati sulla mia tesi triennale

## Requisiti per gli script Python

- pandas
- folium
- geopy
