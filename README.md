# Portfolio – Progetti Digital Humanities

## 🗺️ Tosa Nikki Map Project  
Ricostruzione digitale del viaggio narrato nel *Tosa Nikki* di Ki no Tsurayuki.  
Dal testo originale giapponese sono state estratte alcune tappe del viaggio, convertite in toponimi moderni, geolocalizzate con Geopy, e visualizzate su una mappa interattiva tramite Folium.  
Include un sito web con contenuti storici divulgativi.  
📁 *Cartella: `tosa-nikki-map-project`*

## 🧠 Analisi Pragmatica di *Gone Girl*
